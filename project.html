CASE STUDY / GAME PORTFOLIO PAGE

NEON RAID 界面系统与视觉叙事案例

角色: 视觉设计 / 交互规划 / 内容编排

周期: 4 周

类型: 动作射击 / 赛博世界观包装

CASE FILE

OVERVIEW

目标是让玩家在 10 秒内记住项目气质。

这个内容页示例展示了案例页该如何承接首页的视觉张力。它不是简单堆图，而是把项目拆成目标、 系统、过程和结果，让访问者能快速理解你的设计判断。

“作品集内容页要像任务简报，读完之后，别人知道你做了什么，也知道你为什么这么做。”

SYSTEM

把信息拆成三个层级，读起来才像一套成熟系统。

A1

世界观入口

使用单句式标题作为记忆锚点，让访问者先理解项目的核心气氛，再进入更细的内容。

B2

信息压缩

把职责、周期、工具和成果压成短标签，避免长段说明在第一屏抢走注意力。

C3

节奏控制

用边框、色块和大号章节标题做分隔，让页面在长阅读里仍然保留游戏界面般的节奏感。

CHALLENGE

问题

原始案例资料很多，但缺乏层级。读者会先被大量截图淹没，却抓不到你的核心设计贡献。

APPROACH

方法

用大标题、短标签和章节模块重组内容，让每一段都服务于“项目价值”而不是“资料堆砌”。

RESULT

结果

一个既能承接首页视觉气质，又能清楚展示过程与成果的内容页模板，后续可直接复制到更多项目。
